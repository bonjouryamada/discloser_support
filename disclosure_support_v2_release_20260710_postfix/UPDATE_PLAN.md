# Disclosure Support V2 Update Plan

## 目的

公開中のStreamlitアプリが持つEDINET取得、財務数値入力、3制度比較、Excel出力を維持しながら、
今回整備した全101項目の制度横断データへ更新する。

## 実装済み

- PDF全6区分・101項目を正本データとして採用
- mapping_id 101件、手動レビュー13件を画面表示
- キーワード、区分、手動レビューによる絞り込み
- 東証適時開示、内部者取引規制、臨時報告書の横断表示
- 「該当記載なし」が非該当を意味しない旨の明示
- EDINET取得と財務数値による動的判定ライン算出を維持
- 選択項目単位のExcelレポート出力
- 出典・更新情報ページと公式一次情報リンク
- PC・スマートフォン向けレスポンシブ表示

## 次フェーズ

1. 現行GitHubリポジトリとデプロイ設定を確認する。
2. V2を別URLで先行公開し、実際のEDINET APIキーで結合テストする。
3. 運営者表記、問い合わせ先、利用規約を反映する。
4. 公式一次情報との差分確認日と更新責任者を表示する。
5. 利用者テスト後、現行URLをV2へ切り替える。

## スコープ外

V2は法的判断や提出要否を自動確定しない。案件保存、承認ワークフロー、法改正の自動反映は
将来機能として扱う。
